# SI-CAPEK (Sistem Informasi Catatan Pelaksanaan Kegiatan)

SI-CAPEK adalah aplikasi manajemen kinerja pegawai berbasis **Google Apps Script** yang terintegrasi dengan Google Sheets, Drive, dan Calendar. Aplikasi ini dirancang untuk Puskesmas IV Denpasar Selatan guna memantau logbook kegiatan, pengajuan cuti, dan lembur secara real-time.

## ‚ú® Fitur Utama
- **Input RPK (Rencana Pelaksanaan Kegiatan):** Terintegrasi otomatis ke Google Calendar.
- **Logbook Digital:** Pelaporan kegiatan harian disertai unggah foto dokumentasi ke Google Drive.
- **Manajemen Cuti & Sakit:** Pengajuan mandiri oleh pegawai dengan sistem approval admin.
- **Automated Document Generation:** Generate surat cuti otomatis dalam format PDF.
- **Rekapitulasi:** Export laporan bulanan ke format Excel atau Cetak langsung.
